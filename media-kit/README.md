# Markice Moore Media Kit (8-Page Portrait)

This folder contains a premium, print-ready **editable HTML media kit** designed to export as PDF.

## Files
- `media-kit.html` — master editable layout (8 pages)
- `styles.css` — typography, colors, print styles
- `assets/` — place only Markice Moore's real uploaded images here

## Export to PDF
Open `media-kit.html` in a browser, then print:

- Paper size: **Letter (8.5" × 11")**
- Margins: **None**
- Scale: **100%**
- Background graphics: **On**

This will produce a clean, selectable-text PDF.

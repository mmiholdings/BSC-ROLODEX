# Markice Moore Media Kit (8-page portrait)

This package is a premium, edit-friendly HTML/CSS media kit with selectable text and print-ready page sizing.

## Files
- `markice_media_kit.html` — the full 8-page deck.
- `styles.css` — premium typography, spacing grid, and print styling.
- `assets/` — place only the real uploaded Markice photos.

## Required image files for this version
Use the five uploaded photos from this request and save them with these exact names in `assets/`:

1. `markice_smile_gray.jpg` (smiling gray shirt headshot)
2. `markice_black_leather.jpg` (black leather jacket headshot)
3. `markice_white_jacket.jpg` (white jacket serious portrait)
4. `markice_dog_hat.jpg` (hat + dog lifestyle image)
5. `markice_dog_cafe.jpg` (cafe + dog lifestyle image)

The layout intentionally reuses these real uploaded images across pages to keep visual consistency without introducing any substitute people imagery.

## Export-ready PDF steps
1. Open `markice_media_kit.html` in Chrome/Edge.
2. Print / Save as PDF.
3. Settings:
   - Paper: Letter (8.5 x 11 in)
   - Margins: None
   - Background graphics: On

The exported PDF remains clean with selectable text.

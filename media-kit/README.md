## Export to PDF
Open `markice_media_kit.html` in a modern browser and print/export to PDF:
- Paper size: Letter (8.5 x 11 in)
- Margins: None
- Background graphics: Enabled

The deck uses selectable text and page-break rules for clean PDF output.

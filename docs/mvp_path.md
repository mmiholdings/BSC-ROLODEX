# First Working MVP Path

1. **Live signal ingestion**: ingest `r/forhire`, `r/slavelabour`, and HN posts every 15 minutes.
2. **Qualification/scoring**: accept only simple paid jobs and compute score.
3. **Instant outreach**: generate first response with AI.
4. **Routing**: select playbook and alert Telegram for manual supplier handoff.
5. **Tracking**: persist events and monitor margins.
6. **Learning**: check source close rates daily and adjust thresholds.

## 0-48h money path
- Focus on micro tasks: logo tweak, landing page edit, scraping, list building.
- Route to pre-vetted freelancer templates.
- Keep gross margin rule: `quoted_price >= 1.8 * supplier_cost`.
